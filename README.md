## Welcome to SynthCity 

Welcome to SynthCity, a multiplayer collaborative music-making game! Each player will compose 8 seconds of the song during their turn using a Sequencer built with React and Tone.js. At the end of the game, players can hear their full song! \
\
Check it out at: https://synthcity.herokuapp.com/ \
